2.What are objects in javascript?

Ans:
  • An object is an unordered collection of key-value pairs
  • Each key-value pair is called a property.
  • its consists of properties and methods
  • object have keys which are converted to string types,the properties and method do not follow the insertion order.

4.Explain prototypes?

Ans:
  • /Prototypes are the mechanism by which Javascript objects inherit features from one another.
  • __proto__ object will be created inside the object which will refer to the Prototype of its class constructor or function constructor whenever an object will be created.
  • Using prototype property we can also add more elements to an object which will be accessible by other objects

5.What is prototype chain?

Ans:
  • Objects created using new keyword, inherit from a prototype called Object.
  • All objects in JavaScript are instances of Object.
  • prototype. That means all the objects in JavaScript, inherit the properties and methods from Object.prototype. This is called Prototype chaining.
